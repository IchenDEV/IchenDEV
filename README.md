# Chen Li / idevlab

Building local-first AI tools, agent infrastructure, and native macOS software.

- Blog: [blogs.idevlab.cn](https://blogs.idevlab.cn)
- Site: [IchenDEV.github.io](https://github.com/IchenDEV/IchenDEV.github.io)

## Current focus

- AI agents that can safely use real tools and services
- native macOS interfaces for everyday LLM workflows
- local-first bridges between cloud platforms and desktop/dev environments

## Recent active projects

Selected public repos I have actively shipped in the last 12 months:

### Native macOS apps

- [OpenType](https://github.com/IchenDEV/opentype) - Local AI-powered voice input for the macOS menu bar, combining on-device ASR, OCR context, LLM cleanup, and voice-command workflows.
- [SkillsUI](https://github.com/IchenDEV/skills-ui) - A native macOS app to browse, install, and organize `skills.sh` packages across coding agents.
- [Skill MCP Manager](https://github.com/IchenDEV/skill-mcp-mgr) - A unified control plane for Agent Skills and MCP servers across Codex, Claude Code, Cursor, Gemini CLI, and more.

### Agent infrastructure

- [Passka](https://github.com/IchenDEV/passka) - A local auth broker for AI agents; long-lived credentials stay in macOS Keychain while agents use short-lived leases and audited proxy access.
- [agentalk](https://github.com/IchenDEV/agentalk) - `curl` for AI agents, bringing the A2A protocol to the terminal.
- [LarkFS](https://github.com/IchenDEV/larkfs) - Mount Lark/Feishu as a local filesystem through FUSE or WebDAV, so cloud resources can be handled like local files.

### Knowledge and skill systems

- [superman](https://github.com/IchenDEV/superman) - A living library of perspective-based agent skills distilled from top thinkers into reusable cognitive tools.
